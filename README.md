🎓 Udemy Clone Website

A simple Udemy-inspired course platform UI built using HTML and CSS.
This project replicates the basic layout and design of the Udemy homepage to practice frontend web development fundamentals.

🚀 Project Overview

This project is a static Udemy clone website that displays course sections such as recommended courses, topics, popular courses, and a footer.

It was created to practice:

HTML page structure

CSS layout and styling

Responsive grid layouts

UI design principles

The website is fully built with basic HTML and CSS only without using JavaScript or frameworks.

🛠️ Technologies Used

HTML5 – For creating the structure of the website

CSS3 – For styling and layout

Flexbox & Grid – For course card alignment and layout

Google Fonts – For typography
