# Udemy Clone Website

This project is a simple Udemy homepage clone created using **HTML and CSS**.  
The goal of this project is to practice basic frontend web development and understand how course websites are structured.

## Technologies Used
- HTML5
- CSS3

## Features
- Recommended courses section
- Topics you might like section
- Most popular courses section
- Course cards with image, title, rating, and price
- Simple footer layout

## Project Purpose
This project helped me learn:

- HTML page structure
- CSS styling and layout
- Creating course card UI
- Building a basic static website

## Future Improvements
- Add JavaScript for interactivity
- Make the website responsive
- Add search functionality
